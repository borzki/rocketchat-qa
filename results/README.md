# Experimental test output

This directory holds raw results from Assignment 3 experiments. Files are
regenerated by the scripts in [`tests/`](../tests/) and should NOT be hand
edited.

| Path | Produced by | Description |
|---|---|---|
| `performance/smoke-summary.json` | `npm run perf:smoke` | k6 smoke-run summary |
| `performance/load-summary.json` | `npm run perf:load` | k6 load-run summary |
| `performance/stress-summary.json` | `npm run perf:stress` | k6 stress-run summary |
| `performance/spike-summary.json` | `npm run perf:spike` | k6 spike-run summary |
| `performance/endurance-summary.json` | `npm run perf:endurance` | k6 endurance-run summary |
| `mutation/mutation-report.html` | `npm run mutation` | Stryker HTML report (interactive) |
| `mutation/mutation-report.json` | `npm run mutation` | Stryker machine-readable report |
| `chaos/probe-*.jsonl` | `node tests/chaos/scenarios/<name>.js` | Per-scenario 2-second probe stream |
| `chaos/chaos-summary.json` | `node tests/chaos/analyze.js` | Aggregated availability / MTTR |

See [`docs/assignment3/assignment3-report.md`](../docs/assignment3/assignment3-report.md)
for the full analysis.
